## Data Type

- Data Type determines
    1. What type of data a Variable can Hold
    2. How much of Storage/Space should be allocated in Memory for that Variable

## Char

- Initially ASCII found out that
    - there are totally 128 Characters which are most commonly Used
    - It Inlcudes â†’ All special Characters, a-z, A-Z, 0-9 etc..
    - So, to represent these 128 characters â†’ 8 bits are Required â‡’ 1 Byte
    - i.e., Languages which follow ASCII will require 1 Byte of Memory for Char data type
    - char â†’ 1 Byte [ like C, C++ ]
- Later IEEE found out that
    - there were more than 128 Characters [ like Hindi chars, Telugu chars, Chinese chars etc.. ]
    - These chars are around 65536 chars & are termed as Unicode Chars [ as it follows UTF standards ]
    - So, to represent these 65536 characters â†’ 16 bits are Required â‡’ 2 Bytes
    - i.e., Languages which follow UTF will require 2 Bytes of Memory for Char data type
    - char â†’ 2 Byte [ like Java ]

## Note:

- When an Opereation is involved between numbers,
    - Java will implicitly type convert that result to highest type
        - i.e., the Reason, `byte c = a + b;` gets error
        - As 10+20 = 30 is converted to `Int` type & we are assigning `int` to `byte` type
- Resultant of a Relational Operations is Always a `Boolean` type
- Take Care of **Short-Circuit** during evaluation of Realational AND & OR operations
    - `(i > j) || (++i < ++j) || (i++ < j++)` where i=5 & j=10
