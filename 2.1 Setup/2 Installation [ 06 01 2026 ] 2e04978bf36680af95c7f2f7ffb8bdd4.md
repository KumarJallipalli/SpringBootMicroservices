## JDK Installation [ on Ubuntu Linux ]

1. Using Tar File
    - Download the Tar file from OpenJDK
    - Extract it
        - right click & extract
    - Move it to usr folder
        - `sudo mv /home/sj/Downloads/jdk-25.0.1/ /usr/local/java/`
    - **Set the `JAVA_HOME` environment variable**
        - `nano ~/.bashrc`
        - Add these lines at the end of file
            - `export JAVA_HOME=/usr/lib/jvm/<extracted_directory_name>`
            - `export PATH=$JAVA_HOME/bin:$PATH`
        - Reload config file → `source ~/.bashrc`
    - Check Versions
        - `java --version`
        - `javc --version`
2. Using Deb File
    - Download the .deb file from Oracle Website
        - go to folder containing the deb file
        - Run Command `sudo apt install ./jdk-21_linux-x64_bin.deb`
    - Check Versions
        - `java --version`
        - `javc --version`

## How to RUN Java Files

- Create a File → `Hello.java`
    - Add some working code & save it
    
    ```java
    class Hello {
        public static void main (String[] args) {
            System.out.println("Hello JDK..!");
        }
    }
    ```
    
- Open Terminal → `ctrl + ``
- Commands
    1. Compile java file → `javac Hello.java`
    2. Execute java file →  `java Hello`
